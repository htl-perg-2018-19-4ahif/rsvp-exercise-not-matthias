# RSVP - REST API


A simple implementation of a rsvp rest api.

## Installation

```
git clone https://github.com/htl-perg-2018-19-4ahif/rsvp-exercise-not-matthias.git
cd rsvp-exercise-not-matthias
npm install
```

## Usage

### REST API
```
npm run api
```

### Website
```
npm run web
```

### Both
```
npm start
```
